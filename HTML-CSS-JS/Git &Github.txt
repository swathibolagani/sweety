Actual process:
1) git init: it will initiate git into your folder.
2) git add: it will add all the files into git.
3) git commit -m "updated" : it will commit your git and added a message update.

we get error if we did frist time: to solve that error do following commands:
                
                git config --global user.email "your email id"
                git config --global user.name  " username"
after this try git add . and git commit -m "updated" again
4) git remote add origin "link"
5) git push -u origin master
